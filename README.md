# Live Stock Market Dashboard

A professional financial dashboard built with Python and Streamlit.

## Features
- Live stock prices with KPI cards
- Candlestick charts with SMA, EMA, Bollinger Bands
- Technical indicators: RSI, MACD
- Financial metrics: PE, ROE, ROA, Margins
- Company information and logo
- Latest news with sentiment analysis
- Stock comparison tool
- Portfolio simulator with Sharpe Ratio
- Market overview with top gainers/losers
- CSV export on every page

## Technologies
- Python, Streamlit, yfinance, Plotly
- Pandas, NumPy, scikit-learn, TextBlob

## Installation
```
pip install -r requirements.txt
streamlit run dashboard.py
```
